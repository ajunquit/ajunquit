## Perfil

Soy **Ingeniero de Software** con experiencia desarrollando aplicaciones empresariales para banca, aerolíneas, facturación electrónica, gestión documental, GRC y sistemas financieros. Me desempeño como desarrollador **hands-on** y también como referente técnico, participando en diseño de arquitectura, implementación, revisiones de código, automatización, pruebas y despliegues.

Mi propuesta de valor está en combinar profundidad técnica con visión end-to-end: construir software mantenible, resolver problemas complejos en producción, modernizar plataformas críticas y dejar bases sólidas para que los equipos escalen con calidad.
